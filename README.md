## Features
Interactive Search: Users can search for any city to retrieve current weather conditions.

5-Day Forecast: Provides a visual breakdown of upcoming weather trends.

Persistent Search History: Uses localStorage to save recently searched cities for quick one-click access.

Modern UI: Built with Bootstrap for a polished, responsive design that works on desktops, tablets, and phones.

Custom Modals: Replaces browser alerts with sleek Bootstrap modals for error handling and user notifications.

## User Story
As a frequent traveler,
I want to see the current weather and a 5-day forecast for a specific city,
So that I can plan my trips and dress appropriately for the conditions.

## Acceptance Criteria
[ ] Search for a city and see current weather (temp, humidity, wind speed).

[ ] View a 5-day forecast with icons representing weather conditions.

[ ] Click on a city in the search history to re-search that city.

[ ] Responsive design ensures usability on mobile devices.

[ ] Error handling via modals if a city is not found or the API fails.
